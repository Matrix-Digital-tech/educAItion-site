# educAItion Marketing Website

AI literacy camp pilot marketing site for families — kids ages 8–12 and parents/guardians.

## What's Here

Static site for the first educAItion AI literacy camp pilot in Chicago (August 17–21, 2026):

- **Landing** — "AI is changing our future. Together, we can shape it."
- **About** — Our beliefs, principles, generational connection
- **Curriculum** — 5-day preview (kids-only, parent-only, family-together sessions)
- **Register** — Netlify Form with privacy notice and explicit consent

## Deploy

### Netlify
1. Connect this repo to Netlify
2. Base directory: leave blank
3. Publish directory: `/`
4. Deploy

### GitHub Pages
1. Settings → Pages → Source: Deploy from a branch
2. Branch: `main`, folder: `Root`
3. Save

## Crawl Blocking
This site blocks all crawlers and AI models via `robots.txt` and `<meta name="robots" content="noindex, nofollow">`.

## Domain
Custom domain: **educaite.me** — configure via Netlify DNS.
